Alinhamentos de elementos e conteúdo usando Flexbox

Flexbox é um mecanismo do CSS3 que permite aplicar facilmente o alinhamento e distribuição de elementos e conteúdos mo HTML.

Na prática, é uma técnica diferente para solucionar problemas e minimizar o grande uso de paddings, margins, floats, positions na resolução de dificuldades na construção de layouts.

As propriedades do flexbox são os seguintes:

  • flex-direction: Define qual a direção o container deseja empilhar os itens (column, column-reverse, row, row-reverse).
  • flex-wrap: Específica se os itens flexíveis devem ser encapsulados ou não (flex-wrap: wrap, nowrap, wrap-reverse).
  • flex-flow: Propriedade abreviada para defiir flex-direction e flex-wrap (flex-flow: row wrap).
  • justify-content: Alinhamento de intens (justify-content: center, flex-start, flex-end, sapce-around, space-between
  • align-itens: Propriedade é usada para alinhar os itens flexíveis.
  • align-content: Propriedade é usada para alinhar as linhas flexíveis.
